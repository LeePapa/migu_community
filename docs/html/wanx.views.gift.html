wanx.views.gift.gifts_exchange(*args, **kwargs)¶

Uri:

/gifts/exchange
参数:
  • requ_type –requ_type = 'get_judge'：检查兑换是否满足条件，requ_type = 'pay_exchange'：付费礼物兑换，requ_type = 'gold_exchange'：游米礼物兑换
