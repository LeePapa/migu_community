wanx.views.activity.comment module¶

wanx.views.activity.comment.activity_comments(*args, **kwargs)¶

获取活动评论 (GET)

Uri:

/activity/<string:aid>/comments/
参数:
  • maxs – 最后时间, 0代表当前时间, 无此参数按page来分页
  • page – 页码(数据可能有重复, 建议按照maxs分页)
  • nbr – 每页数量
返回:

{‘comments’: list, ‘end_page’: bool, ‘maxs’: timestamp}
